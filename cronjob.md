# cronjob

## Overview

* the cron daemon runs scheduled tasks at a specific time

## Place Scripts within the Following Folders

```bash
/etc/cron.daily/
/etc/cron.hourly/
/etc/cron.weekly/
/etc/cron.monthly/
```

## Place Commands in the Cronjob \(Cron Table Software to Edit Schedule\)

### **List Cronjobs**

```bash
crontab -l
```

## Edit or Create New Cronjobs

```bash
crontab -e
```
